Development Roadmap

We'll treat this like a real engineering project and build one milestone at a time:

Project setup (Python virtual environment, Git, Docker, folder structure)
Docker Compose (PostgreSQL + Airflow running)
Weather API extraction (requests)
Transform JSON using pandas
Generate CSV
Load data into PostgreSQL
Add logging and error handling
Create the Airflow DAG
Test scheduled execution
Production improvements (retries, configuration, monitoring)

This approach ensures you understand every layer instead of copying a finished project.
